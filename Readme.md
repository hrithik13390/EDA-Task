# Exploratory Data Analysis (EDA)

## Objective

This project performs Exploratory Data Analysis (EDA) on a CSV dataset using Python.

## Technologies Used

- Python
- Pandas
- Matplotlib

## Dataset

dataset.csv

## Analysis Performed

- Data Loading
- Data Inspection
- Summary Statistics
- Missing Value Detection
- Data Visualization

## How to Run

```bash
pip install -r requirements.txt
python eda.py
```
